# Outreach tracker

Six cold-email curators + four aggregator submissions. Markdown table, diff-able
in git, copy-pasteable into journalist replies. Six rows do not justify a UI.

Update `Sent at` (UTC ISO), `Status`, `Response`, and `Follow-up by` (UTC date)
in-place as each item moves.

Status vocabulary: `not sent` → `sent` → `replied` / `featured` / `no reply` /
`declined`.
